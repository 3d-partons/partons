Compute GPD with some input parameters.

Parameters
_x: Bjorken variable
_xi: longitudinal momentum
_t: momentum transfer (Mandelstam variable)
_MuF: Factorisation
_MuR: Re-normalisation
_gpdComputeType: H, Ht, E, Et, ... or ALL. See GPDComputeType for more details.
Returns
Return results in an GPDOutputData class.
Contains GPD results for each flavor of quarks and for each GPDs (H, Ht, E, Et, ...) if computable.

Detailed Description

Defines existing quark flavors and associates them with an enum type.


It represents the dependence on quark flavors of hadron structure distributions. Current values are:

  • UNDEFINED: empty field
  • UP, DOWN, STRANGE, CHARM, BOTTOM, TOP: explicit quark names
